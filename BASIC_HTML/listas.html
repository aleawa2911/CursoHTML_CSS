Listas

Las listas se usan para almacenar cosas en ellas, con las etiquetas < ol > para listas ordenadas, < ul > para listas desordenadas y < li >para los items de la lista.

Lista Ordenada

Las listas ordenadas, son como su nombre lo dice, listas, que llevan un orden, sin CSS, el navegador las lee como 1. tal, 2. tal, 3. tal, con usos como pasos de un proceso, pero eso da igual, ya que con CSS le vamos a dar el formato de todos modos, PERO, es importante para el SEO, posicionamiento y para los lectores de accesibilidad.

Ejemplo de una lista ordenada.

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Lista Desordenada

Las listas desordenadas, al contrario de las ordenadas, simplemente son elementos agrupados en una lista, sin importar el orden, como barras laterales, o caracteristicas de algo.

Ejemplo de una lista desordenada.

  • Elemento 1
  • Elemento 2
  • Elemento 3
