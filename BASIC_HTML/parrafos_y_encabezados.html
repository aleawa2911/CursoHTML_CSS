Explicación de estructura, encabezados y parrafos

Estructura básica de html

  • <!DOCTYPE html> le indica al navegador la version de html que vamos a usar.
  • <html> indica que empieza a escribirse el codigo.
  • <head> indica el head, es decir la info no visible de la pagina, que incluye cosas como el titulo(title) que aparece en la pestaña.
  • <body> indica el body de la pagina, es decir la info visible.

Encabezados

Encabezados, son usados en los titulos o subtitulos, en forma de arbol, es decir 1-2-3-4-5-6, es importante solo tener UN h1, por temas de posicionamiento y lector para personas ciegas, estos abarcan las siguientes etiquetas:

  • < h1 >
  • < h2 >
  • < h3 >
  • < h4 >
  • < h5 >
  • < h6 >

Ejemplo de cada encabezado:

  1. Header 1

  2. Header 2

  3. Header 3

  4. Header 4

  5. Header 5
  6. Header 6

Parrafos
