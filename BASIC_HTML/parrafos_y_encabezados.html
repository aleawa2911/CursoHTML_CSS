Explicación etiquetas y ejemplos

Encabezados

Encabezados, son usados en los titulos o subtitulos, en forma de arbol, es decir 1-2-3-4-5-6, es importante solo tener UN h1, por temas de posicionamiento y lector para personas ciegas, estos abarcan las siguientes etiquetas:

  • < h1 >
  • < h2 >
  • < h3 >
  • < h4 >
  • < h5 >
  • < h6 >

Ejemplo de cada encabezado:

  1. Header 1

  2. Header 2

  3. Header 3

  4. Header 4

  5. Header 5
  6. Header 6

Parrafos

Como su nombre lo dice, son parrafos, bloques de texto. Se utiliza la etiqueta < p >. Este mismo texto es un ejemplo de parrafo.
