FORMULARIOS

Los formularios son la forma en que se recibe la informacion del usuario, pueden tener inputs de los siguientes tipos:

text

Reciben texto plano.

email

Reciben correos electronicos, exigen una @ para permitir enviar el texto.

password

Reciben contraseñás, ocultandola detras de simbolos.

number

Selecciona numeros, se puede limitar con min y max.

checkbox

Para marcar opciones multiples

Para marcar una sola opción.

file

Para subir archivos, puede limitarse el tipo de archivo con el atributo accept.

button

Botón normal.

reset

Para resetear los datos escritos.

tel

Para numeros de telefono.

url

Para enlaces.

search

Campo de busqueda.

range

Para seleccionar un rango con un slider.

color

Para seleccionar un color.

submit

Botón para enviar el formulario.
