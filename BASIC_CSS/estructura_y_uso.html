CSS, explicación y aplicación

Como funciona CSS?

CSS aplica estilos (propiedades) a elementos HTML mediante selectores, siguiendo esta ruta: selector → propiedad → valor.

Formas de aplicar CSS

Para aplicar CSS en una pagina web, hay 3 formas distintas, de las cuales hay una que es la que vamos a usar siempre, ya que es la mejor práctica y es la que se usa en el entorno profesional, es decir, usar un archivo externo de css.

CSS en archivo externo, styles.css:

Para aplicarlo de este modo, debemos seguir los siguientes pasos:

  1. Crear el archivo correspondiente a css, es decir, styles.css, suele estar en la raiz del proyecto, dentro de una carpeta llamada public.
  2. Para aplicarlo a una pagina, en el head, añadiremos la etiqueta link, la cual llevara rel="stylesheet" para decirle a html que la conexion sera para usar el archivo como hoja de estilos, y href que es la ubicacion del mismo.
  3. Al elemento deseado, le aplicaremos una clase o id para poder usarlo en el archivo de css
  4. En el archivo css, utilizaremos selectores para dar propiedades css a los elementos html que querramos segun su clase o id.

CSS inline:

Se aplicarán los estilos solo a la linea, no es una práctica correcta, talvez solo para pruebas rapidas, pero en la vida real, es mejor usar un archivo de estilos vinculado en el head

Ejemplo: <p style="color: red;">hola<p/>

CSS interno:

CSS interno quiere decir que se añadirá una etiqueta <style> <style/> en el archivo html.

Ejemplo:

<html>
<style>
</style>
</html>
