Importar con enlace

En head, linkeamos el enlace proporcionado por GoogleFonts y ya podemos usarla.

Registrar local con @font-face

  1. Descargamos la fuente deseada
  2. La guardamos en la carpeta de fuentes del proyecto
  3. Creamos fonts.css para conectar las fuentes con @font-face
  4. Linkeamos font.css en los archivos del proyecto
  5. Utilizamos las fuentes de forma normal
