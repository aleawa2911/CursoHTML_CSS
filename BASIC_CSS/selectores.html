Selectores, clases y id's

Selectores es la forma de seleccionar elementos de un documento html en la hoja de estilos.

Selección por elemento.

Se selecciona con el nombre de un elemento, se utiliza de la siguiente manera:

p{propiedad:valor}

Selección por clase.

Se selecciona por medio de una clase, se aplican los estilos a todos los elementos que pertenezcan a la misma:

.clase-seleccionada{propiedad:valor} --------- Este es el metodo mas usado, ya que es reutilizable, permite combinar clases, es mas facil de mantener y ocasiona menos conflictos.

Selección por id.

Similar a las clases pero se selecciona por medio de id, con la diferencia de que es una seleccion de un unico elemento:

#id-seleccionado{propiedad:valor}
